## Fitness Tracker Application

This is a full-stack web application designed to help users track their fitness activities, monitor progress, and set goals.

### Features

*   **User Management:** Secure registration and login using JWT authentication. Users can view and update their profile information.
*   **Workout Logging:** Users can log their workouts, including details like exercise type, duration, intensity, and personal notes.
*   **Dashboard Monitoring:** A dashboard provides workout history and progress towards goals.
*   **Goal Setting:** Users can set fitness goals (e.g., run a certain distance, complete a number of workouts) and track their status.

### Tech Stack

*   **Backend:** Spring Boot (Java), Spring Security, Spring Data JPA
*   **Frontend:** React, Javascript, Redux Toolkit, Vite, shadcn/ui
*   **Database:** H2 (In-Memory for development/testing)
